# Weather App (Python Tkinter)

This is a simple desktop weather application built using Python and Tkinter.

## Features
- Fetches real-time weather data
- User-friendly interface
- Displays temperature, AQI and weather conditions

## Tech Used
- Python
- Tkinter
- Weather API

## Note
This project was initially built with the help of AI, and I am currently improving my understanding of the code and enhancing the features.
